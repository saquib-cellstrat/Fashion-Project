# Project Skills Index

This directory contains project-level Cursor skills used to keep AI behavior consistent in this repository.

## Structure

- `_shared/`: reusable templates and writing conventions
- `<skill-name>/SKILL.md`: one skill per directory

## Naming Rules

- Use lowercase kebab-case for skill directory names.
- Keep each `SKILL.md` concise and task-focused.
- Include YAML frontmatter with:
  - `name`
  - `description`

## Required `SKILL.md` Outline

Use a predictable structure for easier maintenance:

1. `# Skill Name`
2. `## When to use`
3. `## Required workflow`
4. `## Quality checks`
5. `## References`
